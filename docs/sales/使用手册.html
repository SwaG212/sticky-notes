1. 下载与安装

1 下载 Setup.1.0.0.exe（约 91MB）
2 双击运行，选择安装目录，勾选"创建桌面快捷方式"
3 安装完成后，桌面会出现「便利贴」图标
4 双击启动，或设为开机自启后自动运行
提示：软件启动后出现在系统托盘（右下角小图标区域），不会弹出窗口。按 Alt + ` 呼出。

2. 首次配置（必须）

首次启动会自动弹出设置窗口。如果没弹出，右键系统托盘图标 →「配置 API Key」。

1获取 DeepSeek API Key：打开 platform.deepseek.com，注册账号 → 左侧「API Keys」→ 创建新 Key → 复制
2 在设置页粘贴到「API Key」输入框
3「API 地址」保持默认 https://api.deepseek.com（无需改动）
4 点击右上角「确认」保存
注意：DeepSeek 注册即送免费额度，日常使用够用数月。API Key 使用系统级加密存储，不会上传任何服务器。

3. 便利贴页 — 记录与整理任务

这是主页面，快捷键 Alt + ` 唤出后默认显示。

3.1 输入文字

在底部输入框打字，可以是单条任务，也可以是多行杂乱想法。例如：

"明天前把合同出了，找运维要服务器账号，记得买牛奶"

3.2 粘贴截图

复制任意截图后，在输入框中 Ctrl + V 粘贴。支持粘贴最多 3 张图片，单张不超过 10MB。图片会显示在输入框上方。

3.3 AI 整理

点击「整理 ▸」按钮，或直接关掉窗口（失去焦点自动整理），或按 Ctrl + Enter。

AI 会：

  • 先 OCR 识别截图中的中文文字
  • 将文字 + 识别结果一起发给 DeepSeek AI
  • AI 拆解为独立任务，返回结构化清单
离线可用：没有网络时，系统会按换行符自动拆分文字为独立任务，不会丢失数据。

3.4 任务操作

操作方法
勾选完成点击任务前面的圆圈
编辑文字双击任务文字，修改后按回车确认
删除任务鼠标移到任务上，点击右侧出现的 ✕
拖拽排序鼠标按住未完成任务右侧的 ⋮⋮ 手柄，上下拖动
设置提醒鼠标移到任务上，点击时间（如 --:--），选择小时和分钟

3.5 定时提醒

给任务设置闹钟时间后，系统每 60 秒检查一次，到点弹出提醒窗口（右下角弹窗，始终置顶）。点击「确认」关闭。

  • 每天凌晨会自动清除所有未触发的闹钟时间（恢复为 --:--）
  • 昨日未完成任务会自动迁移到今天
  • 已过时间不会弹窗

4. 记事本页 — 写笔记

切换方式：便利贴页点击右下角「→」按钮，或按 Alt + 2。

4.1 文件列表

点击左上角「☰」打开文件列表：

  • 搜索：在顶部搜索框输入关键词，按回车过滤文件名
  • 切换笔记：单击文件名
  • 重命名：双击文件名，输入新名称后回车
  • 删除：右键文件名（两次，防误触），删除后放入回收站
  • 置顶：点击文件名右侧 📍 按钮，置顶后显示 📌，置顶笔记排在列表上方并有浅灰背景

点击文件列表外部区域或再次点击「☰」关闭列表。

4.2 创建新笔记

点击左上角「+」按钮。新笔记默认命名为 untitled_1.md。

4.3 AI 自动命名

当你写完笔记并切换到其他文件（或关闭窗口）时，如果笔记文件名为 untitled_X.md 且有内容，AI 会读取内容并自动生成 15 字以内的文件名。

  • 打开笔记 → 写字 → 切换到其他笔记 → AI 自动改名
  • 如果没有内容（空文件），系统会自动删除该空白文件

4.4 自动保存

每次输入后 300 毫秒自动保存，关闭窗口时也会最终保存。无需手动保存。

4.5 返回便利贴页

点击左下角「←」按钮，或按 Alt + 1。

5. 日报生成

在便利贴页底部点击「日报」按钮。系统会：

将今日任务按"已完成在前、未完成在后"排序，拼接为"姓名 + 日期 + 编号任务清单"格式，自动写入剪贴板。

然后直接 Ctrl + V 粘贴到微信、钉钉或其他地方即可。

自定义姓名：在设置页填写「日报姓名」，不填则使用系统用户名。

6. 设置页详解

打开方式：切换到记事本页 → 点击右上角 ⚙ 按钮，或右键系统托盘图标 →「配置 API Key」。

设置项说明
API KeyDeepSeek 的 API 密钥，必填。加密存储，安全
API 地址默认 DeepSeek 官方，可改为其他兼容接口
日报姓名日报中显示"某某 0619"的名字
文件读取位置笔记存储目录。留空用默认（AppData），可改为 D:\我的笔记 等
开机自启勾选后随 Windows 启动，使用系统登录项
快捷键点击输入框，按下想要的组合键即可自定义（见下方）
修改文件读取位置后，笔记列表会立即刷新为新目录内容，原目录文件不会被删除。

7. 快捷键一览

功能默认快捷键可自定义
唤出/隐藏窗口Alt + `✔
AI 整理 (在便利贴页)Ctrl + Enter✔
切换到便利贴页Alt + 1✔
切换到记事本页Alt + 2✔

所有快捷键都可在设置页自定义：点击对应输入框 → 按下新组合键 → 自动记录。按 Esc 取消。

8. 系统托盘

便利贴常驻系统托盘（右下角通知区域）。黄色小方块图标。

操作效果
左键单击图标唤出/隐藏窗口
右键单击图标弹出菜单

右键菜单功能：

  • 打开便利贴 — 显示窗口
  • 配置 API Key — 打开设置页
  • 开机自启 — 勾选/取消
  • 退出便利贴 — 彻底关闭程序

9. 常见问题

Q: API Key 无效 / 401 错误？

检查 Key 是否复制完整，是否有多余空格。去 DeepSeek 后台确认 Key 状态。

Q: API 余额不足 / 402 错误？

DeepSeek 免费额度用完了，去 platform.deepseek.com 充值（很便宜，几块钱够用几个月）。

Q: 为什么粘贴截图后没反应？

确认粘贴的是图片（不是文件路径）。单张图片不超过 10MB。最多同时 3 张。

Q: 笔记存在哪里？

默认在 %APPDATA%/sticky-notes/notes/。可在设置页改为任意目录。

Q: 任务数据存在哪里？

%APPDATA%/sticky-notes/tasks/YYYY-MM-DD.json。按日期分文件，旧任务不会丢失。

Q: 快捷键不生效？

检查是否被其他软件占用。Alt + ` 在某些键盘上可能需要按 Alt + ~。可在设置页改为其他组合键。

Q: 怎么彻底卸载？

Windows 设置 → 应用 → 找到「便利贴」→ 卸载。卸载后手动删除 %APPDATA%/sticky-notes/ 清理数据。

Q: 窗口位置能调吗？

窗口固定在屏幕右下角（360×500），不支持拖动。即呼即走的定位就是这个设计思路。
