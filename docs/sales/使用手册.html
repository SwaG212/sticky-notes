一页看完所有功能怎么用

怎么唤出来

按 Alt + `（ESC 下面那个键），窗口从右下角弹出。用完点窗口外面或再按一次就隐藏。

软件启动后在右下角系统托盘常驻，是个黄色小方块。左键点它也能唤出。

开机自动启动，不用每次都手动开。

便利贴页 — 记待办

打字记

底部输入框直接敲字，多件事可以换行写。

贴截图

截图后 Ctrl+V 粘贴，内置 OCR 识别图中文字。

一键整理

点「整理 ▸」或直接关窗，AI 自动拆成任务清单。

✓ 勾掉任务

点圆圈勾选完成，自动沉底，动画丝滑。

✎ 编辑文字

双击任务文字就能改，回车确认。

✕ 删除

鼠标移到任务上，右侧出现 ✕，点了就删。

⏰ 设闹钟

鼠标移到任务 → 点时间（--:--）→ 选几时几分。到点右下角弹窗提醒。

⋮⋮ 拖拽排序

按住 ⋮⋮ 手柄上下拖，想怎么排就怎么排。

📋 一键日报

点「日报」按钮，今日任务自动写入剪贴板，Ctrl+V 直接粘贴到微信/钉钉。

每天凌晨自动把昨天没完成的任务搬到今天，闹钟时间清零。不丢东西。

记事本页 — 写笔记

便利贴页点右下角「→」切换过去（快捷键 Alt+2）。回来按 Alt+1 或点「←」。

☰ 文件列表

左上角 ☰ 打开，列出所有笔记。点文件名切换，双击改名，右键删除。

📌 置顶

文件列表中点 📍 图标置顶，重要笔记始终排在最前面。

🔍 搜索

文件列表顶部搜索框，输入关键词回车过滤。

+ 新建

点左上角 +，创建空白笔记。

🤖 AI 自动起名

写完笔记切到别的文件，AI 自动根据内容生成文件名。

💾 自动保存

打字即存，关窗也存。不用找保存按钮。

设置 — 改配置

切换到记事本页 → 点右上角 ⚙。或者右键系统托盘图标 →「配置 API Key」。

  • API Key — DeepSeek 密钥，没它 AI 不能用
  • 日报姓名 — 日报里显示"张三 0619"
  • 文件位置 — 笔记存哪，不改就默认 AppData
  • 快捷键 — 点输入框按下新组合键即改
