便利贴 · 配置指南

装好后花 2 分钟搞定，以后就不用管了

第一步：获取 DeepSeek API Key

1打开 platform.deepseek.com，注册账号（用手机号或邮箱）
2登录后点左侧菜单 「API Keys」
3点 「创建新的 API Key」，随便起个名字（比如"便利贴"），点创建
4复制生成的 Key（一长串 sk- 开头的字符），只显示一次，立刻复制
注册就送免费额度，日常够用好几个月。充值也很便宜，几块钱用一年。

第二步：填入便利贴

1打开便利贴（系统托盘右键 →「打开便利贴」或按 Alt+`）
2点击右上角 ⚙ 齿轮图标（在记事本页），或右键系统托盘图标 →「配置 API Key」
3把刚复制的 Key 粘贴到 「API Key」 输入框
4「API 地址」不用动，保持 https://api.deepseek.com
5点右上角 「确认」，搞定
Key 是以加密方式存你电脑上的，不会上传到任何服务器，放心。

其他可配置项

设置项干什么的不填会怎样
日报姓名日报里显示你的名字（如"张三 0619"）用 Windows 用户名
文件读取位置笔记存哪。比如改到 D:\我的笔记默认 AppData 目录
开机自启勾上后开机自动启动需手动打开
快捷键四个全局快捷键，点击输入框后按下新组合键即改用默认的

默认快捷键

干什么按什么
唤出 / 隐藏Alt + `
AI 整理Ctrl + Enter
切到便利贴Alt + 1
切到记事本Alt + 2
快捷键在设置页点一下输入框再按新组合键就能改。比如把 Ctrl+Enter 改成 Ctrl+E。

常见报错

报错怎么办
「请先配置 API Key」还没填 Key，按第二步操作
「API Key 无效」(401)Key 复制漏了或有多余空格，重新复制粘贴
「余额不足」(402)DeepSeek 免费额度用完了，去 platform.deepseek.com 充值
快捷键没反应被其他软件占了，去设置页换一个组合键
AI 返回格式异常偶尔网络抖动，重试一次就好
